
### libjs
Проект с заданиями по работе с готовыми библиотеками js:
- [Choices.js](https://github.com/Choices-js/Choices) - библиотека для кастомизации выпадающего списка с большим количеством настроек и возможностями для стилизации в CSS.  
- [Swiper](https://swiperjs.com/) - библиотека, которая позволяет создавать слайдеры практически любого типа: галерею, ленту товаров или публикаций и так далее.
- [JustValidate](https://just-validate.dev/) - библиотека для проверки введённых данных в текстовых полях формы и отображения подсказок при нарушении правил ввода.
- [JustValidate](https://just-validate.dev/) - библиотека для проверки введённых данных в текстовых полях формы и отображения подсказок при нарушении правил ввода.
- [SimpleBar](https://github.com/Grsmto/simplebar) - библиотека для создания прокрутки больших текстов.

### filmbook
Блокнот по сохранению информации о фильмах.
- форма ввода с валидацией полей
- сохранение в локальное хранилище
- возможность удалять/изменять сохраненные фильмы
- сортировка сохраненных фильмов

### webFilmbook
UI блокнота просмотренных/не просмотренных фильмов с сохранением фильмов на сервере.
